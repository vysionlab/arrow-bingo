# Arrow Bingo PowerPoint

A PowerPoint-based Jeopardy-style game focused on life safety topics.

## How to Play

1. Open the fixed PowerPoint in presentation mode
2. Click a point value on the game board → question appears (answer hidden)
3. Click/tap again → answer reveals
4. Click "Return to Board" to go back and select another question

## Files

- `Life Safety Jeopardy Game.pptx` - Original game file
- `Life Safety Jeopardy Game - Fixed.pptx` - Updated/fixed version (recommended)
